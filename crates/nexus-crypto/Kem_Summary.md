# Nexus Crypto KEM Summary

## Scope

This summary covers the key encapsulation and randomness pieces of `nexus-crypto`.

## Module Map

| Module | Current responsibility |
| --- | --- |
| `mlkem.rs` | ML-KEM key generation, encapsulation, decapsulation |
| `csprng.rs` | random byte source helpers |
| `traits.rs` | KEM-facing trait contracts |

## Important Facts

- ML-KEM is part of the validator key bundle generated by `nexus-keygen`.
- Randomness helpers rely on `getrandom`.

## Minimal Read Paths

- KEM flow: `src/lib.rs` ‚Üí `mlkem.rs`
- Entropy path: `src/lib.rs` ‚Üí `csprng.rs`
